# Deprecated CUDA templates

This directory keeps legacy native CUDA templates that are no longer part of the active build path.

- `ffpa_attn_fwd_templates_sm90.cuh`: experimental SM90 TMA forward path.
- `ffpa_attn_bwd_templates.cuh`: legacy native CUDA backward path.

Current development focuses on Triton kernels. Keep these files as historical references only.
